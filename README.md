# MEA Trace Viewer

Lightweight Tkinter GUI for visualizing HD-MEA `.brw` / HDF5 recordings.

## Features

- Spatial electrode grid based on physical channel layout
- Click-to-select electrodes
- Embedded matplotlib trace viewer
- Dual-handle time-range slider
- Dark-mode UI
- Save plots as PNG/PDF/SVG

## Dependencies

```bash
pip install numpy matplotlib h5py
```
